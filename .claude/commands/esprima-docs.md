Search and reference the Esprima documentation when working with JavaScript/TypeScript parsing.

Esprima is a high performance, standard-compliant ECMAScript parser written in ECMAScript.

**Documentation**: https://docs.esprima.org/en/latest/

**Key sections**:
- Syntax Tree Format (ESTree specification)
- Lexical Analysis (tokenization)
- Syntactic Analysis (parsing)
- API Reference

When using esprima for parsing:
1. Parse TypeScript/JavaScript to ESTree AST
2. Walk AST nodes
3. Transform nodes to Python equivalents
4. Generate Python code

Use this skill to look up:
- AST node types and structure
- Parsing options
- How specific JavaScript constructs are represented in the AST
- ESTree specification details
